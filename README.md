# Sales-Data-Analysis-Using-ChatGPT

### Overview
In this project, we use ChatGPT and Prompt Engineering to utilize it's LLM-based powers to analyze the Sales Data and Create an Interactive Dashboard.

### Objectives
1. Convert the excel data to csv format.
2. Convert the csv to json for ChatGPT tp analyze it.
3. From the json, give ChatGPT 2 rows of data and ask it to give detailed analysis of the data and suggest 5 visualization metrics.
4. Ask ChatGPT to create an html dashboard that sits on your local computer and has an option to upload the json file to create the dashboard for the particular data.
